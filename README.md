# Integration-of-information-systems
Комплекс лабораторних робіт з дисципліни "Інтеграція інформаційних систем". Проєкт: Вебсистема для управління каталогом книг бібліотеки (React, FastAPI, MongoDB).
# 📚 Information System for Library Book Catalog

Репозиторій для виконання комплексу лабораторних робіт з дисципліни **"Інтеграція інформаційних систем"**.

## 📝 Про проєкт
[cite_start]Метою роботи є створення вебсистеми для автоматизації управління каталогом книг у бібліотеці[cite: 69]. [cite_start]Система замінює застарілі паперові каталоги та Excel-таблиці, забезпечуючи зручний пошук, облік книг та взаємодію з читачами[cite: 68, 89].

## 🛠 Технологічний стек
[cite_start]Проєкт реалізовано як вебдодаток з використанням сучасних інструментів[cite: 83, 110]:

* [cite_start]**Front-end:** React, Tailwind CSS, Axios[cite: 113].
* [cite_start]**Back-end:** Python, FastAPI[cite: 111, 112].
* [cite_start]**База даних:** MongoDB Atlas[cite: 103, 112].
* [cite_start]**Інструменти:** VS Code, Postman, GitHub[cite: 115, 116].

## ✨ Функціональні можливості
[cite_start]Система підтримує дві ролі користувачів: **Адміністратор** та **Читач**[cite: 230].

### Для користувачів (Читачів):
* [cite_start]**Реєстрація та авторизація** (email/пароль)[cite: 227, 228].
* [cite_start]**Пошук та фільтрація** книг за назвою, автором, жанром та роком видання[cite: 223, 224].
* [cite_start]**Перегляд деталей** книги (опис, обкладинка, рейтинг)[cite: 221].
* [cite_start]**Бронювання** книг та залишення відгуків[cite: 96, 239].

### Для адміністраторів:
* [cite_start]**Управління каталогом:** додавання, редагування та видалення книг[cite: 91].
* [cite_start]**Інвентаризація:** контроль наявності книг на полицях[cite: 94].
* [cite_start]**Аналітика:** перегляд статистики активності та популярності книг[cite: 93].

## 📂 Структура бази даних
Дані зберігаються у хмарній базі MongoDB Atlas та включають колекції:
[cite_start]`Користувачі`, `Книги`, `Автори`, `Видавництва`, `Відгуки`, `Бронювання` [cite: 233-240].
